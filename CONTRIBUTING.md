# Contributing

Thank you for improving gPlug Energy Cockpit.

1. Create a focused branch from `main`.
2. Keep source changes inside `custom_components/gplug_energy`.
3. Test initial setup and the **Configure** options flow.
4. Verify the dashboard card in a current Home Assistant installation.
5. Open a pull request and describe the affected source entities and units.

Power source sensors may use W or kW. Energy source sensors may use Wh, kWh,
or MWh. Public integration sensors must continue to expose power in kW and
energy in kWh.
